Kinds of Computer Accessories

Below are the groups they belong

Input & Output

An input device transmits data to a computer system for processing, and an output device displays or reproduces the results of that processing. Input devices only allow data to be entered into a computer, whereas output devices only receive data from another device.An output device can receive data from another device and use that data to generate output, but it cannot send data to another device.

Storage

A storage device is any type of computing hardware used to store, transfer, or extract data files and objects. Storage devices can temporarily and permanently hold and store data. They can be inside or outside of a computer, server, or computing device.

Communication

Communications device is any piece of hardware that can transmit data, instructions, and information between a sending and receiving device. A modem is a type of communications device that connects a communications channel to a sending or receiving device, such as a computer.
